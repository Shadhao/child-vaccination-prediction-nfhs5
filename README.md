# Machine Learning-Based Prediction of Child Vaccination Status in India (NFHS-5)

## Overview

This project applies machine learning techniques to predict childhood vaccination status in India using data from the National Family Health Survey (NFHS-5). The study compares multiple classification algorithms and identifies key socioeconomic, demographic, and healthcare-related determinants influencing vaccination coverage.

## Objectives

* Predict childhood vaccination status using demographic and healthcare variables.
* Compare the performance of multiple machine learning models.
* Identify key determinants influencing vaccination coverage.

## Dataset

* Source: National Family Health Survey (NFHS-5), India
* Sample Size: 12,377 children aged 12–23 months

## Models Implemented

* Logistic Regression
* Ridge Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Random Forest
* XGBoost

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* ROC-AUC

## Key Findings

* XGBoost achieved the best predictive performance among the evaluated models.
* Maternal education, household wealth, antenatal care visits, institutional delivery, and health worker visits were among the most important predictors of vaccination status.
